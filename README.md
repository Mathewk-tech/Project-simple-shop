# Simple Shop
This is a project submission for phase-1 javascript module. The project is about creating a webpage that allows people to buy products from Simple shop.The simple shop website allows user's to do the following

- To view the available products.

- To select the product they want by clicking add to cart,which automatically calculates the total price for the products selected.

### Project setup
The required tools are:

- visual studio code
- node version 18
- livesever extension

### How to run:

- clone the project using
```
https://github.com/Mathewk-tech/Project-simple-shop.git
```
- Open the folder of the cloned project using vs code.
- Open the folder of the challenge you want to run.
- Right click on its index.html and choose open wtih live server.- Then on the pop up that appears on the bottom right click "open in browser."

## What You'll Learn

This project provides hands-on experience with fundamental web development concepts and modern JavaScript frameworks. By building this simple e-commerce application, you'll gain practical knowledge in:

- **Front-end Web Development**: Creating responsive user interfaces using HTML, CSS, and JavaScript
- **React Fundamentals**: Building reusable components, managing component state, and handling user interactions
- **JavaScript Programming**: Working with arrays, objects, event listeners, and DOM manipulation
- **E-commerce Logic**: Implementing shopping cart functionality, including adding/removing items and calculating totals
- **Payment Integration**: Understanding how to integrate third-party payment services (M-Pesa) into web applications
- **Data Handling**: Fetching and displaying product data from JSON sources
- **User Experience**: Designing intuitive interfaces for product browsing and checkout processes
